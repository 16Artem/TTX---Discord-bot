Модерация

Основные команды модерации

Команда Описание Использование Требуемые права
/ban Заблокировать пользователя на сервере /ban @пользователь [причина] BAN_MEMBERS
/kick Исключить пользователя с сервера /kick @пользователь [причина] KICK_MEMBERS
/mute Ограничить пользователя в отправке сообщений /mute @пользователь [время] [причина] MANAGE_ROLES
/warn Выдать предупреждение пользователю /warn @пользователь [причина] MANAGE_MESSAGES
/clear Удалить сообщения в канале /clear [количество] MANAGE_MESSAGES
/lock Закрыть канал для отправки сообщений /lock MANAGE_CHANNELS
/unlock Открыть канал для отправки сообщений /unlock MANAGE_CHANNELS

Настройки модерации

Команда Описание Использование Требуемые права
/automod Включить/выключить авто-модерацию /automod [on/off] ADMINISTRATOR
/logchannel Установить канал для логов /logchannel #канал ADMINISTRATOR
/warnings Посмотреть предупреждения пользователя /warnings @пользователь MANAGE_MESSAGES
